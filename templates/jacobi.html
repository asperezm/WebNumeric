Jacobi Data

--------------------||--------------------

In order to operate properly, the jacobi functions:

- Ax = b

- (D-L-U)x=b

- Dx-(L+U)x=b

- Dx=(L+U)x+b

- x=D^-1*(L+U)x+D^-1*b

- x=Tj*x+Cj

- Where Tj = D^-1*(L+U) and Cj=D^-1*b

--------------------||--------------------


The Jacobi method is an iterative method to solve systems of linear equations of type AX = b. This method uses formulas like fixed point iteration. The objective of this method is to build a convergent sequence iteratively defined. The limit of this succession is the solution of the system. For practicity, the algorithm is stopped after a number of steps to arrive at an approximation to the value of X of the system solution. The sequence is constructed by decomposing the matrix of system A in the following way: A = D + R
