Secant Method

--------------------||--------------------

In order to operate properly:

-The function must be continuous on the interval

-The root must be in the interval

-The derivative does not become 0 in the root

Strategies for faster convergence:

-Take as initial approximations the interval where the root is

-If you used one of the fixed-point methods, such as Newton, you can take the same initial approach and modify the figure minimally for the second value

--------------------||--------------------


The Secant method can be thought of as a simplification of Newton-Raphson's method. Instead of taking the derivative of the function whose root we want to find, it approaches by a line secant (hence the name) to the curve, whose slope is approximately equal to that derived in the starting point. The main difference with the previous method is to know two points of the function for to be able to generate such a line. Let \(x_0\) y \(x_1\) belonging to a certain \(f(x)\) can be defined: \[x_{n+1} = {x_{n}} - { f(x_n) (x_n - x_{n-1}) \over f(x_n) - f(x_{n-1})}.\]
