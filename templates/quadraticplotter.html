    Strategies for convergence:

  • Not necessarily more points implies a better approximation.
    • Given a function \(f\) in \([a, b]\) and nodes \( a = x_0 \lt x_1 \lt \cdots \lt x_n = b \) , an interpolating linear spline \(p\) for \(f\) is a function that satisfies the following conditions:
  • \( p(x) \) is a linear polynomial, defined \(p_j(x)\), over the interval \([x_{j-1}, x_{j}]\) for each \(j=1,...,n\)
  • \( p_j(x_{j-1}) = f(x_{j-1})\) and \(p_j(x_{j}) = f(x_{j})\) for each \(j=1,...,n\)
  • \( p_{j+1}(x_{j}) = p_j(x_j)\) for each \(j=1,...,n-1\)
  • \( p'_{j+1}(x_{j}) = p'_j(x_j)\) for each \(j=1,...,n-1\)
  • \( p''_1(x_1)=0 \)
