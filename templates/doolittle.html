In linear algebra, the Doolittle matrix factorization is an LU decomposition which decomposes a matrix into a lower triangular matrix L and an upper triangular matrix U. The Doolittle matrix decomposition method returns a lower triangular matrix L with ones in it’s diagonal, in such a way that A=LU. It is a form of LU factorization of matrices, which consists of finding a square matrix A, the upper triangular matrix (U) and the lower triangular matrix (L). In this way, L * U = A For the factorization of this case it must be fulfilled that Lii = 1, with i = 1… .., n. This means that all numbers on the diagonal of the lower triangular matrix (L) must be equal to 1.
